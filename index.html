pico.geisel.tech

A learning experiment running on a Raspberry Pi Pico W

This site is hosted entirely from a Raspberry Pi Pico W sitting on my desk. I'll keep adding little test items every now and then. Things I'm testing are available below. Not everything will work completely right all the time I'm sure, but that's part of the process!

RTC

The RTC onboard the Pico W is polled at page load.

  • Time at page load: {rtc} CST
  • Date at page load: {date}

Other

  • Your IP Address: {ip}
  • Onboard tempurature: {temp}

Webpage features

  • The HTML document is stored on and served by the Pico W
  • Styling is done from an external CSS file stored on and served by the Pico W
  • The background image (assuming available screen size) is remotely pulled from https://geisel.tech
